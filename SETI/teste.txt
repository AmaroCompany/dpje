UNSUPPORTED KEY TYPE NULL
Boa tarde,

Solicitamos que siga os passos abaixo para sanar o problema informado:

1.Retire o token da porta USB do computador.
2.Acesse o painel de controle do computador.
3. No painel de controle, acesse as opções de Internet.
4. Selecione a aba “Conteúdo” e logo após “Certificados”
5. Clique no (s) certificado (s) e selecione remover.
6. Após o procedimento, realize o acesso ao sistema que o problema estará
sanado.

Permanecemos à inteira disposição.


FETCH TO SOURCE
Boa tarde,

Informamos que este erro está relacionado com sua conexão de internet e é gerado quando o navegador utilizado não permite a conexão para acesso do PJe, será necessário verifique as configurações de conexão do navegador, do Firewall do Windows e do Antivírus utilizado. O PJeOffice utiliza a porta 8800 para receber dados do PJe (web), portanto deverá verificar se esta porta está tendo o acesso bloqueado por alguma das opções mencionadas.

Permanecemos à inteira disposição.
